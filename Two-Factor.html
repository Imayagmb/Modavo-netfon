Two-Factor Authentication - 2FA

Autenticação de dois fatores.

O Two factor authentication é um procedimento de segurança
que garante que serão necessários 2 fatores únicos para
liberação de uma ação. O primeiro fator é a senha que o
usuário e o segundo pode ser autenticado via token, via
detecção de impressão digital, reconhecimento facial,
código enviado via sms, entre outros.

Funcionamento 2FA

Acesso

O usuário acessa o site
ou aplicativo e digita a
senha cadastrada para
entrar em seu perfil
u completar uma transação.

Recebimento

A Telecall recebe a
tentativa de login e
solicita que o usuário
insira seu número de
telefone para autorizar
o acesso.

Confirmação

Após inserir seu número,
a Telecall envia para o
usuário por SMS,
chamada ou e-mail
um código PIN
de uso único.

Conclusão

O usuário insere o código
no site ou aplicativo para
concluir o processo de verificação.





O 2FA permite que você:

Envie uma senha via SMS, voz
ou e-mail para autenticação do usuário.

Adicione uma camada extra
de segurança além da senha pessoal.

Ofereça maior segurança
para usuários.

2FA Benefícios

Flexibilidade

Enviar OTP por SMS, em caso de falha,
enviar OTP por chamada de voz,
em caso de outra falha,
enviar por e-mail.

Reduz Fraudes

Reduz casos de fraudes e
invasões e evita o acesso
a dados por invasores.
